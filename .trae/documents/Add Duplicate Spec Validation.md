# 烘焙布工具重复规格检测

根据您的要求，我已在计算逻辑中增加了**重复规格检测**功能。

## 功能说明
当您点击“**开始计算最优方案**”按钮时，系统会立即检查您输入的所有规格：
1.  **检测逻辑**：如果发现有两条规格的**长**和**宽**完全一致（例如第1行填了 `30x30`，第3行又填了 `30x30`）。
2.  **提示方式**：会弹出一个轻量级的黑色提示框（Toast），告知您具体是哪一个规格重复了（例如：“规格 #3 (30x30cm) 重复，请合并或修改”）。
3.  **阻断计算**：检测到重复后，系统会暂停计算，等待您修改或合并后再试。

**代码变更**：
-   `js/stats.js`：在 `calculateLayout` 函数中插入了去重校验逻辑。

此功能可有效防止因误输入导致的重复计算。请刷新页面，尝试输入两个相同的规格并点击计算按钮进行测试。
