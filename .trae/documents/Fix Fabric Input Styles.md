# 烘焙布工具输入框样式修复

针对您反馈的“文字被挡住”的问题，这主要是因为在较窄的屏幕上，输入框内部的提示文字（placeholder，即“长”、“宽”）与右侧的单位（cm）重叠，或者与输入的数字冲突。

## 修复内容
1.  **移除内部占位符**：
    *   删除了成品规格输入框中的 `placeholder="长"` 和 `placeholder="宽"`。
    *   **原因**：因为输入框本身很窄，而外部的布局结构（左侧是序号，中间是两个框中间有个 x，右侧是单位）已经非常清晰地表明了这是长宽尺寸，因此内部的文字提示是多余且造成视觉干扰的。

**代码变更**：
-   修改了 `js/stats.js` 中的 `renderSpecs` 函数。

现在输入框应该看起来更加清爽，输入数字时也不会有文字重叠的现象了。请刷新页面查看。
