# 烘焙布工具 PDF 导出功能实现报告

根据您的需求，我为烘焙布裁剪工具添加了**“导出所有方案 (PDF)”**的功能。

## 功能说明
1.  **一键导出**：在“开始计算”按钮下方，新增了一个白色按钮。点击后，系统会将当前计算出的所有排版方案（不仅仅是当前显示的这一个）整理到一个 PDF 文件中。
2.  **内容完整**：PDF 中的每一页对应一个排版方案，包含：
    *   方案序号和利用率
    *   原布尺寸
    *   清晰的排版图（包含切割线和剩余布料）
    *   详细的规格统计文字
3.  **下载路径**：
    *   **电脑/Web端**：直接触发浏览器下载，保存到您的默认下载文件夹。
    *   **安卓 App 端**：严格参考了“库存管理工具”的数据备份逻辑，文件将尝试保存到：
        `Android/data/plus.H5F9023DE/downloads/`
        如果该目录无法访问，则会自动降级为普通的系统下载方式。

## 技术实现
-   引入了 `jspdf` 库来生成 PDF 文件。
-   复用了底层的 Canvas 绘图逻辑，确保导出的 PDF 清晰度高（使用了 5 倍缩放渲染）。
-   文件命名格式为 `Fabric_Layouts_YYYYMMDD.pdf`。

**代码变更**：
-   `src/11_page_fabric.html`：添加按钮和库引用。
-   `js/stats.js`：实现了 `exportAllSolutionsToPDF` 核心逻辑。

请刷新页面，计算一组方案后，尝试点击“导出 PDF”按钮体验。
