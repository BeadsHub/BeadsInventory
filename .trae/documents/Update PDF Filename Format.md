# PDF 文件名格式调整

根据您的要求，我修改了 PDF 导出文件的命名规则。

## 新的命名格式
文件将按照以下格式自动命名：
**`原布长x原布宽_N规格M方案-时间戳.pdf`**

例如：`100x100_3规格5方案-20260125_1430.pdf`

**格式说明**：
*   **原布长x原布宽**：如 `100x100`（注：由于 Windows 系统文件名不支持 `*` 号，我用 `x` 代替）。
*   **连接符**：使用下划线 `_` 代替了中文逗号，以确保在所有设备（包括部分老旧安卓系统）上的文件兼容性。
*   **规格数量**：自动统计当前参与计算的有效规格种类数。
*   **方案数量**：导出文件中包含的排版方案总数。
*   **时间戳**：精确到分钟，格式为 `YYYYMMDD_HHMM`。

**代码变更**：
-   `js/stats.js`：更新了 `exportAllSolutionsToPDF` 函数中的 `fileName` 生成逻辑。

请刷新页面后再次点击“导出所有方案 (PDF)”按钮，查看下载的文件名是否符合预期。
